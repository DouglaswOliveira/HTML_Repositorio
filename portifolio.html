Meu Portfólio Acadêmico

Trabalhos Acadêmicos obtive meu primeiros passos

SQL e Primeiros Códigos

Minha experiência com SQL começou com a aprendizagem dos fundamentos da linguagem de consulta estruturada. Aprendi a criar, manipular e consultar bancos de dados usando comandos SQL. Isso me permitiu entender como os dados são armazenados e gerenciados em um ambiente de banco de dados.

Python e Primeiros Comandos

Aprendi Python, uma linguagem de programação de alto nível conhecida por sua simplicidade e eficiência. Comecei com os comandos básicos, como operações matemáticas, controle de fluxo e manipulação de strings. À medida que avançava, comecei a explorar bibliotecas Python mais complexas, como NumPy e Pandas, que são essenciais para a análise de dados.

Treinamento de Perceptron e Estudo de Machine Learning

Em seguida, mergulhei no campo do Machine Learning, onde treinei meu primeiro modelo de Perceptron. O estudo do Perceptron me permitiu entender os fundamentos das redes neurais e como elas podem ser usadas para resolver problemas complexos. Também explorei outros algoritmos de Machine Learning e técnicas de pré-processamento de dados.

Trabalho em HTML e CSS

Finalmente, desenvolvi habilidades em desenvolvimento web, aprendendo HTML e CSS. Aprendi a criar páginas web estáticas usando HTML e a estilizá-las usando CSS. Isso me deu uma compreensão sólida de como os sites são construídos e como eles podem ser projetados para serem visualmente atraentes e fáceis de usar.
